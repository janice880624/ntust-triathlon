間歇訓練說明

間歇訓練（Interval Training）是一種交替進行高強度運動和低強度恢復的訓練方式。這種訓練可以有效提升心肺功能，並在短時間內燃燒更多熱量

請選擇項目
跑步間歇
游泳間歇
騎車間歇

@ I pace ≈ 是你 5 公里比賽的速度，或約 VO₂max 配速

@ M pace ≈ 你馬拉松比賽能維持的平均配速，通常是最大心率的 75–84% 左右

M pace 1 km x n

熱身:
  • 慢跑10分鐘
  • 100m x 4 @28s 間休1min
主課表:
  • 1km @M pace、間休1min
  • 以上重複 n 趟
緩和:
  • 慢跑5分鐘

M pace 2 km x n

熱身:
  • 慢跑10分鐘
  • 100m x 4 @28s 間休1min
主課表:
  • 2km @M pace、間休2min
  • 以上重複 n 趟
緩和:
  • 慢跑5分鐘

I pace 400m x n r 1.5mins

熱身:
  • 慢跑10分鐘
  • 100m x 4 @26s 間休1min
主課表:
  • 400m x n @I pace
  • 每趟跑完原地休1.5分鐘
緩和:
  • 慢跑5分鐘

@ T pace ≈ 是你快但還能撐 20–30 分鐘的速度，心率區大概是最大心率的 83–88%

100m 間歇組合 x n

熱身:
  • 200 輕鬆游
  • 50 踢水
  • 50 夾浮板划手
  • 100 輕鬆游
主課表:
  • 3 x 300m @ T pace +3秒(間休20秒)、注意姿勢穩定
  • 休1分鐘
  • 以上重複 n 趟
緩和:
  • 200m不限泳姿輕鬆游

Progression 400m x n

熱身:
  • 輕鬆游200m
主課表:
  • n x 400m 間休20秒、速度一開始輕鬆游、一趟比一趟快，最後一趟用9成力
  • 8 x 25m 用最高划頻、(不用太在乎動作細節)，訓練神經肌肉快速反應能力。每趟間休20秒（或是緩游 25m ）。
  • 【如果是50m】8 x 50m 前25m用最高划頻、(不用太在乎動作細節)，訓練神經肌肉快速反應能力。後 25m 緩游。
緩和:
  • 各種泳姿輕鬆游200m

300m + 100m x n 間歇組合

熱身:
  • 100 輕鬆游
  • 50 x 2 踢水
  • 50 x 2 夾浮板划手
  • 100 輕鬆游
主課表:
  • 3 x 300m @ T pace +3秒(間休20秒)、注意姿勢穩定
  • 休1分鐘
  • n x 100m @ T pace (間休15秒)
  • 休1分鐘
緩和:
  • 200m不限泳姿輕鬆游
